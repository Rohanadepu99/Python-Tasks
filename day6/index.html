Research on Python String & List Built-In Methods

1. String Built-In Methods

Strings in Python are immutable sequences of characters, and they come with a variety of built-in methods to perform different operations. Here are some commonly used string methods:

  • lower(): Converts all characters in the string to lowercase.
    • text = "HELLO"
result = text.lower()  # Output: 'hello'
  • upper(): Converts all characters in the string to uppercase.
    • text = "hello"
result = text.upper()  # Output: 'HELLO'
  • split(): Splits a string into a list where each word is a list item.
    • text = "hello world"
result = text.split()  # Output: ['hello', 'world']
  • strip(): Removes leading and trailing whitespace from the string.
    • text = "  hello  "
result = text.strip()  # Output: 'hello'
  • replace(old, new): Replaces occurrences of a substring with another substring.
    • text = "hello world"
result = text.replace("world", "Python")  # Output: 'hello Python'
  • find(substring): Returns the index of the first occurrence of a substring or -1 if not found.
    • text = "hello world"
result = text.find("world")  # Output: 6
  • startswith(prefix): Checks if the string starts with the specified prefix.
    • text = "hello"
result = text.startswith("he")  # Output: True
  • endswith(suffix): Checks if the string ends with the specified suffix.
    • text = "hello"
result = text.endswith("lo")  # Output: True
  • join(iterable): Joins a list of strings into a single string, separated by the string on which the method is called.
    • words = ["hello", "world"]
result = " ".join(words)  # Output: 'hello world'
  • count(substring): Returns the number of occurrences of a substring in the string.
    • text = "hello hello world"
result = text.count("hello")  # Output: 2

2. List Built-In Methods

Lists are mutable sequences that can store multiple items of different types. Python provides a variety of built-in methods to work with lists:

  • append(item): Adds an item to the end of the list.
    • lst = [1, 2, 3]
lst.append(4)  # Output: [1, 2, 3, 4]
  • extend(iterable): Adds all elements of an iterable (e.g., another list) to the end of the list.
    • lst = [1, 2]
lst.extend([3, 4])  # Output: [1, 2, 3, 4]
  • insert(index, item): Inserts an item at the specified index.
    • lst = [1, 3]
lst.insert(1, 2)  # Output: [1, 2, 3]
  • remove(item): Removes the first occurrence of the specified item.
    • lst = [1, 2, 3]
lst.remove(2)  # Output: [1, 3]
  • pop(index): Removes and returns the element at the specified index.
    • lst = [1, 2, 3]
removed_item = lst.pop(1)  # Output: 2, lst = [1, 3]
  • sort(): Sorts the list in place.
    • lst = [3, 1, 2]
lst.sort()  # Output: [1, 2, 3]
  • reverse(): Reverses the order of elements in the list.
    • lst = [1, 2, 3]
lst.reverse()  # Output: [3, 2, 1]
  • index(item): Returns the index of the first occurrence of the specified item.
    • lst = [1, 2, 3]
result = lst.index(2)  # Output: 1
  • count(item): Returns the number of occurrences of the specified item in the list.
    • lst = [1, 2, 2, 3]
result = lst.count(2)  # Output: 2
  • clear(): Removes all elements from the list.
    • lst = [1, 2, 3]
lst.clear()  # Output: []

Conclusion

String Methods provide functionality for transforming, searching, and manipulating strings. Some common operations include changing case, splitting, replacing substrings, and checking string conditions (e.g., startswith, endswith).

List Methods are useful for managing and modifying list data. They allow for adding, removing, and modifying items, as well as sorting and reversing the list.

By understanding and using these built-in methods, Python developers can efficiently perform a wide range of operations on both strings and lists.
